# 🏡 Advanced Loan Simulator (住宅ローン比較シミュレーター)

PythonとStreamlitを利用して構築された、固定金利と変動金利を比較・分析するためのインタラクティブな住宅ローンシミュレーターです。

単純な月々の支払額の計算だけでなく、**「将来の金利上昇リスクのスケジューリング」**や**「月々の支払可能額からの借入可能限界額の逆算」**に対応しており、より現実的かつ高度な資金計画の立案をサポートします。

## ✨ 主な機能 (Features)

* **動的な金利変動スケジュールの設定**
    * 変動金利プランにおいて、「何年目に金利が何%に上がるか」というシナリオをリスト形式で複数追加・編集できます。金利上昇時の支払額の跳ね上がり（リスク）を視覚的に確認可能です。
* **2つの強力な計算エンジン**
    * **Tab 1: 支払額からの逆算 (Affordability Engine)**
        * 「毎月〇〇万円なら払える」という希望支払額と金利変動シナリオから、現在価値（PV）を用いて**いくらまで借りられるか（元本）**を逆算します。
    * **Tab 2: 借入金額からの正算 (Forward Simulation)**
        * 借入希望額から毎月の返済額の推移を計算し、金利変動に伴う負担増を折れ線グラフで可視化します。
* **リアルタイムなグラフ描画**
    * 入力値を変更すると即座に再計算が行われ、結果がUIとグラフに反映されます。

## 🛠️ 環境構築 (Installation)

このアプリケーションを実行するには、Python 3.8以上がインストールされている環境が必要です。

1. **リポジトリのクローン（またはスクリプトのダウンロード）**
   ```bash
   git clone <your-repository-url>
   cd <your-repository-directory>

```

2. **必要なライブラリのインストール**
```bash
pip install streamlit pandas numpy plotly

```



## 🚀 実行方法 (Usage)

以下のコマンドをターミナルで実行して、アプリケーションを起動します。

```bash
streamlit run app.py

```

コマンドを実行すると、自動的にブラウザが立ち上がり（デフォルト: `http://localhost:8501`）、シミュレーターをご利用いただけます。

### 使い方ガイド

1. **サイドバー（共通設定）:**
* 変動金利の「初期金利」を設定します。
* 「上昇スケジュール設定」の表を使って、将来の金利変更シナリオを入力します（例: `年目: 6`, `新金利: 1.0`）。行の追加・削除も可能です。


2. **メイン画面（タブ切り替え）:**
* **💰 支払額から逆算:** 毎月の予算をもとに物件の予算上限を知りたい場合に使用します。
* **📈 金額から計算:** 既に買いたい物件の価格（借入額）が決まっており、将来の支払推移や総支払額を比較したい場合に使用します。



## ⚠️ 免責事項・注意点 (Disclaimer)

* **計算方式について:** 本シミュレーターは「元利均等返済方式」を前提としています。ボーナス払いや元金均等返済には対応していません。
* **激変緩和措置について:** 日本の多くの金融機関が採用している「5年ルール（金利が変わっても5年間は支払額を変えない）」や「125%ルール（上がっても前の支払額の125%まで）」は、**金利上昇リスクを純粋に可視化するためあえて考慮していません**。金利変更月に即座に支払額が再計算される厳しめのシミュレーションとなっています。
* **諸費用等について:** 実際の住宅ローン契約時に発生する初期費用（事務手数料、保証料、印紙代など）や、購入後の税金（固定資産税等）、維持費は計算に含まれていません。本アプリの計算結果はあくまで目安としてご利用ください。
